# DC 리플레이 채팅 (라이브 반응 플레이어)

**DC 리플레이 채팅**은 과거의 스포츠 경기, E스포츠, 방송 등의 이벤트 당시 디시인사이드 갤러리의 "실시간 반응"을 다시 경험할 수 있게 해주는 도구입니다.

![미리보기](https://via.placeholder.com/800x400?text=DC+Replay+Chat+Preview)

## 주요 기능
- **자동 수집 (Auto Scrape)**: 갤러리 주소와 시간만 입력하면 과거 데이터를 찾아옵니다.
- **실시간 리플레이**: 과거의 글들이 당시 속도에 맞춰 채팅처럼 올라옵니다.
- **히트맵 (Heatmap)**: 반응이 폭발했던 하이라이트 구간을 시각적으로 확인합니다.
- **스마트 컨트롤**: 배속(x2~x10), 점프, 정확한 시간 이동을 지원합니다.
- **보안**: 디시인사이드 도메인만 수집하도록 안전장치(SSRF 방지)가 적용되어 있습니다.

## 사용 방법 (배포 가이드)

대회 심사위원이나 일반 사용자들에게 다음 두 가지 방법 중 하나를 안내하세요.

### 방법 1. 웹 버전 (Vercel) - *가벼운 감상용*
설치 없이 링크만 클릭해서 바로 볼 수 있습니다. **최근 1~2시간 데이터**를 볼 때 추천합니다.
1. [Vercel](https://vercel.com)에 이 코드를 배포합니다.
2. 생성된 링크를 공유합니다.
3. **주의**: 무료 서버 특성상 10초 이상 걸리는(12시간 전 등) 데이터 수집은 끊길 수 있습니다.

### 방법 2. 무설치 PC 버전 (Portable) - *헤비 유저용*
**시간 제한 없이** 장시간의 데이터를 수집하고 싶다면 이 방법을 추천합니다. 설치가 필요 없습니다.
1. `DC_Replay_Portable.zip` (제작된 파일)의 압축을 풉니다.
2. 폴더 안의 **`start.bat`** 파일을 실행합니다.
3. 브라우저가 자동으로 열리며 바로 사용할 수 있습니다.

## 팁: 북마크렛 (간편 실행기)
웹 버전(방법 1)을 사용하는 경우, 아래 코드를 브라우저 북마크에 등록하면 갤러리를 보다가 **클릭 한 번으로** 리플레이를 켤 수 있습니다.

```javascript
javascript:(function(){
  const targetUrl = 'https://YOUR-VERCEL-APP-URL.vercel.app'; 
  const currentUrl = encodeURIComponent(window.location.href);
  window.open(targetUrl + '?url=' + currentUrl + '&auto=true', '_blank');
})();
```
*(위 코드에서 `YOUR-VERCEL-APP-URL`을 실제 배포 주소로 변경하세요)*

## 제작자 가이드 (배포판 만들기)
PC 버전(Portable)을 직접 만들려면 프로젝트 폴더에서 **`create_portable_dist.bat`** 파일을 실행하세요. 자동으로 배포용 폴더가 생성됩니다.
